# Respirar

> Air quality monitor for Colombia — ICA and PM2.5 data visualization.

Real-time and historical air quality index (ICA) and PM2.5 readings across Colombian cities. Built on open environmental data from IDEAM and municipal monitoring networks.

**Stack:** Vite · Vanilla JS · HTML · CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Data source

[IDEAM — Instituto de Hidrología, Meteorología y Estudios Ambientales](http://www.ideam.gov.co)
